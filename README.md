# 테스트 계정 정보

Spring Security JdbcUserDetailsManager를 사용한 테스트 계정

| username | password(문서용) | role |
|---|---|---|
| admin | admin1234! | ROLE_ADMIN |
| manager | manager1234! | ROLE_MANAGER |
| user1 | user1234! | ROLE_USER |
## 역할 설명

USER
- 회원 상세 페이지 접근 가능

MANAGER
- 제품 등록 페이지 접근 가능

ADMIN
- 회원 목록 페이지 접근 가능
