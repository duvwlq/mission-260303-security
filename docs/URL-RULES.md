# URL 접근 권한

| URL | 접근 권한 |
|----|----|
| / | 모든 사용자 |
| /login | 모든 사용자 |
| /product/list | 모든 사용자 |
| /product/add | MANAGER, ADMIN |
| /member/list | ADMIN |
| /member/detail | USER |

## 설명

product/list  
→ 누구나 접근 가능

product/add  
→ 관리자와 매니저만 접근 가능

member/list  
→ 관리자만 접근 가능

member/detail  
→ 일반 사용자 접근 가능
